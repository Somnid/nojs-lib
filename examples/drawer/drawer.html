NoJS Drawer

This works by using adjactency selectors and a checkbox with :checked. A label is wrapped around the content but since we don't want it to trigger when any part is clicked we need to add pointer-events: none. We add pointer-events: all to the menu SVG. The SVG is not wrapped in a button otherwise that would absorb the click event instead of forwarding it to the label and checking the checkbox, but we do need to give it role="button" for screen readers. When the checkbox is triggered, the adjacent panel will slide out. Note that we have to place the panel outside by default, if we were to place it one the screen and transform it out then the animation would trigger on page load.

The overlay is a seperate fixed position div of 100% height and width. We use rgba color instead of opacity because opacity will be animated on page load which we don't want. Also, we need to animate it with the drawer so that it only take up the extra space and animates at the same speed to not leave a gap. This is because clicks on the menu itself should not forward to the overlay or clicking the menu would close the drawer so we need to make sure that it's never under the drawer itself. By default pointer events are off so the overlay will not toggle open, but when activated we turn them on so that clicking off the menu will close it properly.
