NoJS Tab

This works by using adjactency selectors and radio buttons with :checked. Each tab is wrapped in a label so that clicking on the tab button will toggle it (technically clicking the panel does as well but since it's already enabled it doesn't matter). Since radios by definition can only have one toggled element in a group, only one tab can ever be selected. The tab elements are wrapped up in labels which mean it's harder to position them the way we want with the tab strip up top and the single panel at the bottom. We can overcome this with display: contents which will remove the element from participating in layout, so we can slot the elements into the grid defined on the nojs-tab. The titles are helpful if the tab name overflows!
