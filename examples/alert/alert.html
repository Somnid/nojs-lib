NoJS Alert

This works by using adjactency selectors and a checkbox with :checked. The elements are wrapped in a label so that clicking inside causes the checkbox to be toggled. However since we are only concerned with the close button (Ã—), we disabled pointer-events on the rest of the label. This could also be accomplished by moving the element out of the label and having the label itself be the close button. This approach was not selected because it requires the user to supply additional identifiers. Note that the close button is actually not a button, this is because a button would absorb the click event and not propogate it to the checkbox label, so instead we just use the raw SVG but give it an aria role so it still reads like a button. When the user clicks the close button the event is propgated to the label which toggles the checkbox and hides the content. Since the content is hidden and no longer clickable it is effectively removed for the user. One downside to the CSS-only version is that it cannot easily be animated. We could roll it up into a 0 height element or fade it out but depending on what's around it, it may still take up layout space which is undesirable. Animation to display:none is not possible. If we enhanced it with JS we could supply more fancy transitions. We could also allow the alert to be completely removed from the DOM.
